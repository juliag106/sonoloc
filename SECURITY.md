## 报告漏洞

sonoloc 主要处理本地音频与数值计算，不涉及网络服务，安全风险较低。
如果你仍发现了潜在的安全问题（例如解析不可信音频文件时的崩溃、
反序列化配置导致的风险等），请**不要**直接公开 issue，而是通过 GitHub
的私有安全通告（Security Advisory）或私信维护者报告。

请在报告中尽量包含：

- 受影响的版本与运行环境；
- 复现步骤或最小样例；
- 你评估的影响范围。

我们会在合理时间内确认并反馈修复计划。感谢你的负责任披露。
